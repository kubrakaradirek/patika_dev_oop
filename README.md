## 🏛️ Part 2: Object-Oriented Programming (OOP)
*Focusing on system design, software architecture, and the four pillars of OOP.*

This module transitions from "writing code" to "designing systems" by implementing real-world models.

### 🛡️ The Four Pillars Implemented:
1. **Encapsulation:** Protecting data integrity via access modifiers and getter/setter logic.
2. **Inheritance:** Establishing hierarchical relationships to promote code reusability.
3. **Polymorphism:** Implementing dynamic behavior where objects can take multiple forms.
4. **Abstraction:** Simplifying complex systems using Abstract classes and Interfaces.

### 📂 Specialized Modeling Assignments:
- 🏛️ **University Management System:** Designing hierarchical class diagrams and faculty/student relationships.
- 🦁 **Zoo Management:** Modeling diverse biological entities using inheritance.
- ✈️ **Flight Management System:** A complex booking and fleet management architecture.
- 🎬 **Online Movie System:** Managing data security and user permissions within a streaming model.
- 🛗 **Elevator Simulation:** A logic-heavy project focused on state management and real-time triggers.
